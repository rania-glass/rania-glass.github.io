Staff Selection Program

One of the roles of Georgia Tech Housing is to find students suited for an RA position. As an Office Assistant for Georgia Tech Housing, one of my assigned tasks was to create folders for 400+ applicants and fill them with the documents submitted by each applicant. In order to accomplish this as efficiently as possible, I created a Python-based file manipulation program that can create directories and sort files based on CSV input and user specification.
